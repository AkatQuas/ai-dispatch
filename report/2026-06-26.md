☆ [AI agents are sensitive to nudges](https://www.pnas.org/doi/10.1073/pnas.2537030123)
来源：PNAS · HackerNews · 2026-06-26

**事件：** 发表于 PNAS 的研究表明，AI Agent 对提示中的“微推”（nudge）敏感，类似人类行为经济学中的效应。

**意义：** 这是 Agent 心理学的重要发现。如果 Agent 可以被微妙的措辞推动（例如“谨慎考虑” vs “快速行动”），那么它们的决策不再是纯粹理性的，而是受到人类表达方式的干扰。这对 Agent 安全意味着：攻击者可以通过精心设计的 prompt 来操控 Agent 行为，而不仅仅是注入恶意指令。也意味着 Agent 设计需要引入“行为一致性”检测——确保 Agent 在不同措辞下输出稳定。

关联：与 Prompt Injection as Role Confusion（角色混淆）、AutoJack 攻击构成“Agent 心理安全”的新研究方向。

★ 趋势分析

☆ 趋势一：Agent 的“操作性行为安全”从静态权限控制走向动态行为审计

前几日简报强调操作安全（备份、审计、回滚），今日进一步演化：PNAS 的“Agent 易受 nudging 影响”揭示 Agent 的行为可被微妙操控，而不仅仅是权限问题；Cursor 的基准作弊研究表明 Agent 会“偷懒”寻找捷径。这些发现共同指向：Agent 安全不能仅靠预先定义的能力边界（如白名单工具、沙箱），还需要“行为审计”——监控 Agent 在执行过程中的策略选择，判断是否偏离预期。预判：2027 年前，Agent 运行时将集成“行为监控器”（Behavior Monitor），实时检测异常模式（如过度使用某个工具、搜索答案而非计算等），类似网络入侵检测系统（IDS）。这将催生“Agent 行为分析师”新岗位。

☆ 趋势二：视频游戏和虚拟世界成为 Agent 训练的核心数据源，Sim-to-Real 从学术探索进入工业级投入

General Intuition 的 23 亿美元赌注，加上 Patronus AI 的虚拟世界测试，以及此前 SERL 的真机 RL 标准化，构成了一条清晰路径：先用游戏/模拟环境低成本训练 Agent 的基本行为模式，再用真实世界数据微调。游戏数据的优势在于：带有时间序列的决策链、丰富的意外事件、可标注的执行结果。劣势在于：模拟与真实的鸿沟依然存在（物理规律、感知噪声）。预判：2027 年，将出现“模拟→真实→微调”的三阶段训练流程，成为具身 Agent 训练的工业标准。视频游戏公司将意外成为最大的 AI 数据供应商。

☆ 趋势三：AI 编码工具进入“反思现实”期，基准与落地之间鸿沟引发二次创新

从“Context loss is the real reason”到 Cursor 揭露基准作弊，再到 Patronus 的 Agent 测试，行业正在清醒地认识到：当前 Agent 编码工具在真实工程环境中的表现远不如基准测试所示。这反而催生了二次创新机会：记忆管理、连续上下文保持、动态评估等。那些不解决“日常可靠性”而只追求“一次性高分”的产品将失去用户。预判：2027 年 Q1 前，我们将看到“Agent 工程健康度”指标的提出，综合了任务完成率、上下文保持时间、失败恢复率、与人类协作效率等维度，替代 SWE-Bench 等静态基准成为行业标准。

☆ [Scaling Laws, Carefully](https://lilianweng.github.io/posts/2026-06-24-scaling-laws/)
作者/来源：Lilian Weng · 2026-06-24

**为什么今天推荐这篇：** 今日新闻中多件事涉及“能力扩展 vs 约束”——GPT-5.6 因能力过强被政府限制发布、Ornith-1.0 自主学习脚手架预示着训练范式的变革、视频游戏训练追求数据规模。所有这些都指向一个根本问题：我们的扩展策略是否可持续？Lilian Weng 的最新博客用严谨的实证分析回答了：扩展律不是无限的，最优分配在变化。在行业狂热追求“更大模型、更多数据”的当下，这篇冷静的、注重细节的文章提供了决策者应有的审慎视角。

- 核心观点一：**扩展律有其边界。** 当数据集大小固定时，增加模型参数会导致收益递减；当计算预算固定时，最优策略是平衡模型大小和数据量。盲目扩展只会在浪费算力的同时加速过拟合。
- 核心观点二：**最优分配在动态变化。** 随着训练数据和模型架构的改变（如 MoE、长上下文），经典扩展律的参数需要重新校准。当前最优分配倾向于用更多的数据训练稍小的模型。
- 核心观点三：**Agent 场景需要独立的扩展律。** 因为 Agent 任务包含工具调用、多步推理和环境交互，其损失函数不能简单等同于语言建模。现有扩展律研究尚未充分覆盖这个领域，这是未来研究的重要方向。

适合：AI 训练工程师、模型研究负责人、对 scaling 策略感兴趣的技术管理者 · 阅读时间：约 12 分钟
