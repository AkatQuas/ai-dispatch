☆ [谷歌开源 DiffusionGemma：文本生成迈入扩散时代，速度暴涨 4 倍]
来源：Radarai / Simon Willison · 2026-06-11

**事件：** Google DeepMind 开源了 DiffusionGemma 模型，将图像扩散模型的核心思想应用于文本生成，在单块 H100 上达到每秒 1000+ Token 的推理速度，一张 4090 即可本地运行，并采用 Apache 2.0 许可。

**意义：** 这可能是 2026 年最重要的模型架构创新之一。它彻底抛弃了自回归“逐 token 生成”的范式，改为在固定大小的“画布”上并行去噪，从根本上改变了推理的延迟-吞吐量曲线。对于追求实时交互的 Agent 应用、Vibe Coding 和实时视频生成，DiffusionGemma 提供了一条比量化、剪枝更优雅的加速路径。其开放策略也与 Anthropic 的封闭形成鲜明对比，进一步巩固了 Google 在开源生态中的领导地位。

关联：与小米 MiMo 模型近期展示的 1000+ TPS 性能形成呼应，表明行业解决方案正从单一优化 GPU 管线走向全新的架构设计。

☆ 从“模型”到“Agent Harness”的范式跃迁

本周最强烈的信号是 Agent平台层（Harness）成为 AI 产业的核心战场。DeepSeek 首次设立 Agent Harness 研究员岗位、Claude Design 被识别为一个完整的 Agent Harness、Niteshift 以“反 LLM 锁定”获得 700 万美金种子轮，共同指向一个事实：仅靠好模型无法构建商业壁垒，谁能提供最好的工具、工作流和编排能力，谁就能锁定开发者和企业用户。预判：2026 年下半年，AICoding 赛道的竞争将从“哪个模型更强”转向“哪个 Harness 更好用”，并催生一批 Harness 原生创业公司。

☆ 扩散模型正在“殖民”语言模型

DiffusionGemma 的问世标志着文本生成技术路径的根本性分歧。自回归模型一直以“生成质量”为核心卖点，但也导致了高延迟和高推理成本。扩散式文本生成将推理速度提升 4 倍，同时能够在单张 4090 上运行，这对边缘计算、Agent 实时互动、大规模推理部署极具吸引力。后续的关键看点是：当扩散模型在长文本、多轮对话等场景中被充分优化后，是否会替代自回归成为主流？这将是 Meta、Anthropic 和 OpenAI 必须认真考虑的架构决策。

☆ 安全与开放性的张力进入临界点

Anthropic Fable 5 的退让并非故事的结束，而是 AI 安全哲学辩论的高潮。Anthropic 尝试“在用户不知情的情况下限制行为”的做法遭到了社区的一致谴责，这种“家长式控制”不仅傲慢，而且适得其反。OpenAI 的 PRC 报告和欧盟的 AI 透明度法案进一步强化了“安全必须透明”的共识。预判：未来半年，各家 LLM 厂商将面临更严格的透明度审查，“无声降智”将成为不可接受的产品行为，安全和开放需要在社会技术层面重新平衡。
