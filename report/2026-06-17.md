☆ 具身 AI 的“自主科研”范式确立：从“人类写代码”到“Agent 做实验”

NVIDIA ENPIRE 系统的发布是具身 AI 分水岭事件。此前机器人研究的标准流程是：人类研究员构思方案 → 编写控制代码 → 在仿真或真机上验证 → 分析结果 → 迭代。ENPIRE 将这个过程完全自动化——Agent 自主搭建环境、选择算法、运行实验、生成报告。这意味着机器人学的研究效率将从“人力驱动”转向“算力驱动”，类似 AlphaGo 对围棋研究的颠覆。预判：2027 年内，至少 3 家头部机器人实验室（如 Boston Dynamics、1X、Figure）将部署类似的 Agent 自主实验系统，将机器人技能学习速度提升 10 倍以上。这将倒逼机器人硬件厂商开放标准 API 接口，否则硬件将成为 Agent 自主实验的瓶颈。
